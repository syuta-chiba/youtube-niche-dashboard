真似元候補チャンネル (priority) の定点観測。GitHub Actions が 1日4回 (JST 3・9・15・21時ごろ) にデータを取得。日付の区切りはすべて JST（日本時間）。180秒（3分）以下の Shorts は集計から除外（YouTube Shorts の現仕様上限に合わせた基準）。

🔥 急伸ウォッチ（タブ先頭）

全チャンネル横断で「直近3日の合計 +30 以上 または 直近観測で +20 以上」伸びている動画の一覧。日次の伸びを列で表示。HIT はチャンネル相対で判定（普段 = そのchの全動画 views 中央値の何倍か）: 🎯 HIT = 5倍以上（本物の突出） / 🚀 準HIT = 3〜5倍（伸びかけ） / 離陸中 = それ未満（下限500v・動画5本未満のchは絶対1,000v）。勢い: ▲加速 ▽減速 →横ばい ■停止。ここに出た動画は「真似元候補」＝早期参入の窓が開いているサイン。行クリックで横断分析＋🌍市場検証（急伸検知された動画は YouTube 全体での同テーマ HIT率も出る）。

チャンネル別タブ・上段の数字タイル

タイル上の「今日 / 3日 / 7日 / 28日」ボタンで集計期間を切替できる（対象: 登録者の伸び・再生数の伸び・投稿本数。離陸中/HIT は定義固定のため変わらない）。「今日」= JST 0時から現在まで。「N日」= 最新観測から遡ってN日間。観測履歴がN日に満たない場合は「実データ ◯日」と注記が出る。

  • 登録者の伸び — 選択期間の登録者の増分。下の「累計」が現在の登録者数。
  • 再生数の伸び — 同じ期間の「チャンネル全体（直近300本・Shorts除く）の合計再生数」の増分。特定の1本ではなくチャンネル全体が稼いだ再生数。⚠️ 新作公開直後や、301本目以降の古い動画が集計から外れた期間は数字が歪むことがある。
  • 投稿本数 — 選択期間に公開された本編の本数（公開日は YouTube API の UTC 日付基準のため、JST 朝9時までの公開は前日扱いになることがある）。
  • 離陸中 / 🚀準HIT / 🎯HIT — 急伸ウォッチ条件を満たす動画の、ch相対 tier 別の本数（普段の3倍未満 / 3〜5倍 / 5倍以上）。

グラフ・リスト

  • 登録者数 — 累計（線）と1日の伸び（棒）。
  • 1日の再生回数 — チャンネル合計の日次増分。
  • 1日の投稿本数 — 直近60日の投稿ペース。
  • ★直近HIT / 準HIT — 公開14日以内で ch相対しきい値（普段の3倍）以上の動画（時間軸 sweet spot のシグナル）。「普段の◯倍」表示で突出度が分かる。
  • 広告判定バッジ（like ◯%） — 各動画の like 率で機械判定。⚠️広告疑 = 500再生以上なのに like率 0.4% 未満（広告/購入views の典型）。緑 = 健全域。グレー = 500再生未満で判定対象外、または like 数非公開。判定は取得時点の値なので目安（低評価が多い動画も like率は下がる）。
  • 動画別 views 推移 — 直近30日に公開された動画＋全HIT動画の観測履歴。↑↓キーで動画切替、←→でソート切替。
